Jawaban Lengkap Pertanyaan JavaScript

Pengertian JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi lebih interaktif. Dengan JavaScript, kamu bisa membuat elemen-elemen di halaman web bergerak, merespon aksi pengguna, dan menampilkan konten yang dinamis. Singkatnya, JavaScript adalah jantung dari halaman web modern.


            // Menampilkan pesan "Hello, world!" di dalam elemen dengan id "demo"
            document.getElementById("demo").innerHTML = "Hello, world!";

Editor dan Compiler JavaScript

Editor adalah program yang digunakan untuk menulis kode. Compiler JavaScript sebenarnya tidak dibutuhkan karena JavaScript diinterpretasi langsung oleh browser.

  • Editor Populer: Visual Studio Code, Sublime Text, Atom
  • Compiler: Babel (untuk mengubah ES6+ menjadi ES5)

Variabel di JavaScript

Variabel adalah wadah untuk menyimpan data. JavaScript menyediakan tiga cara untuk mendeklarasikan variabel: `var`, `let`, dan `const`.

var

- Dideklarasikan secara global atau fungsional. - Bisa dideklarasikan ulang dan diubah nilainya kapan saja. - Tidak disarankan digunakan dalam JavaScript modern karena dapat menyebabkan masalah scoping. 


                var x = 10;
                x = "Hello"; // Nilai x bisa diubah

let

- Dideklarasikan dalam blok kode (misalnya, dalam loop atau kondisi). - Tidak bisa dideklarasikan ulang, tetapi nilainya bisa diubah. - Lebih baik digunakan daripada `var` karena menghindari masalah scoping yang tidak terduga. 


                let y = 5;
                y = "world"; // Nilai y bisa diubah

const

- Dideklarasikan dalam blok kode. - Tidak bisa dideklarasikan ulang atau nilainya diubah setelah diinisialisasi. - Ideal untuk variabel yang nilainya tidak akan berubah, seperti konstanta. 


                const pi = 3.14;
                // pi = 3.15; // Ini akan menyebabkan error

Kapan Menggunakan Apa?

  • var: Hindari penggunaan `var` kecuali Anda bekerja dengan kode lama.
  • let: Gunakan untuk variabel yang nilainya mungkin berubah dalam waktu yang lama.
  • const: Gunakan untuk variabel yang nilainya tidak akan berubah setelah diinisialisasi.

Tipe Data di JavaScript

Tipe data di JavaScript menentukan jenis nilai yang dapat disimpan dalam sebuah variabel. Berikut adalah beberapa tipe data dasar yang sering digunakan:

Tipe Data Deskripsi Contoh
Number Representasi angka, baik bilangan bulat maupun desimal. let age = 30;
String Urutan karakter, diapit oleh tanda kutip tunggal atau ganda. let name = "John Doe";
Boolean Nilai logika, hanya memiliki dua nilai: true atau false. let isStudent = true;
Null Nilai khusus yang menunjukkan tidak ada nilai. let car = null;
Undefined Nilai khusus yang menunjukkan variabel belum diinisialisasi. let x; // x belum memiliki nilai
Symbol Nilai unik dan tidak berubah, sering digunakan sebagai properti objek. let id = Symbol("id");
Object Koleksi data yang tidak berurutan, terdiri dari pasangan kunci-nilai. let person = { firstName: "John", lastName: "Doe" };

Catatan:

  • JavaScript adalah bahasa yang bersifat *dynamically typed*, artinya tipe data suatu variabel dapat berubah selama eksekusi program.
  • Tipe data `Object` sangat fleksibel dan merupakan dasar untuk membuat struktur data yang lebih kompleks seperti array.
  • Tipe data `Symbol` adalah fitur yang relatif baru dan digunakan untuk membuat properti unik pada objek.

Operator di JavaScript

Operator Aritmatika

Digunakan untuk melakukan operasi matematika.

Operator Deskripsi Contoh
+ Penjumlahan let x = 5 + 3;
- Pengurangan let x = 5 - 3;
* Pengkalian let x = 5 * 3;
/ Pembagian let x = 5 / 3;

Operator Perbandingan

Operator perbandingan digunakan untuk membandingkan dua nilai dan menghasilkan nilai boolean (true atau false).

Operator Deskripsi Contoh
== Sama dengan (perbandingan nilai saja, tanpa memperhatikan tipe data) let x = 5; let y = "5"; x == y; // true
=== Sama dengan (perbandingan nilai dan tipe data) let x = 5; let y = "5"; x === y; // false
!= Tidak sama dengan (perbandingan nilai saja, tanpa memperhatikan tipe data) let x = 5; let y = "5"; x != y; // false
!== Tidak sama dengan (perbandingan nilai dan tipe data) let x = 5; let y = "5"; x !== y; // true
> Lebih besar dari let x = 10; let y = 5; x > y; // true
< Lebih kecil dari let x = 5; let y = 10; x < y; // true
>= Lebih besar dari atau sama dengan let x = 10; let y = 10; x >= y; // true
<= Lebih kecil dari atau sama dengan let x = 5; let y = 5; x <= y; // true

Catatan:

  • Operator `==` akan mencoba melakukan konversi tipe data jika kedua operand memiliki tipe data yang berbeda. Hal ini dapat menyebabkan hasil yang tidak terduga.
  • Operator `===` lebih ketat dan tidak melakukan konversi tipe data. Selalu disarankan untuk menggunakan `===` untuk perbandingan yang lebih akurat.
  • Operator perbandingan sering digunakan dalam struktur kontrol seperti `if`, `else if`, dan `switch` untuk membuat keputusan dalam program.
